## Cómo descargarlas

### Opción A — desde GitHub (web)
1. Abre la carpeta en el PR/rama: `docs/screenshots/`
2. Entra en cada `.png` → botón **Download**.

### Opción B — clonar / pull
```bash
git fetch origin cursor/facturaexterior-brand-e350
git checkout cursor/facturaexterior-brand-e350
# Las imágenes están en:
# invoice-app/docs/screenshots/
```
